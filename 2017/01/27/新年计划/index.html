2017新年计划

发表于 2017-01-27   |     |   本文总阅读量 次

Hello 2017

新的一年即将到来，祝大家鸡年大吉。今年也是本人的本命年，希望新的一年，技术更上一层楼。今年的学习计划也指定好了。

首先让我自己回顾下2016年曾定下的目标

  • 入门前端，学好Html,css,javascript
  • 熟悉前端流行框架bootstrap,jquery
  • 转行进入IT行业成为一名前端开发者
    经过一年的努力，我已经加入前端开发的大部队，成为一名前端er。可以熟练的运用html和css搭建静态页面，虽然自学的时候花了很多时间在JS上，工作后发现还是jquery用的比较多。工作后，虽然项目不多，主要是直播室和公司官网以及直播室后台管理系统，但是技术熟练度大大提升。非常感谢公司的同事对我的帮助，愿意指导我这样的菜鸟。
    随着前端接触的越深，越是感到前端技术路线繁多，框架层出不穷。对于新的一年，该怎么规划，我已经有了自己的计划

2017年，我的技术方向

  • JS更加熟练直至精通
  • 拥抱Ract全家桶，走React技术栈
  • 熟练webpack
  • 更多的移动端项目
  • 更深入的学习Node.js
    以上就是本人的2017的目标，愿新的一年技术更加精湛，成为一名专业高水平的前端开发工程师

鸡年大吉！
